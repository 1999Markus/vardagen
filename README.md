# Vardagen

Template produced during internship at SinnerSchrader Hamburg.

## Getting Started

* mkdir Vardagen
* cd Vardagen
* git pull https://github.com/simonh9134/vardagen.git
* npm install

### Installing

* cd Vardagen
* gulp dev

## Deployment

* cd Vardagen
* gulp build

## Built With

* [Visual Studio Code](https://code.visualstudio.com/)
* [Gulp](https://gulpjs.com/)
